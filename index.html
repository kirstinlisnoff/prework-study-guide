Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element reppresents the visible content shown to the user
  • HTML attributes are always defined in the beginning element.

CSS

  • inline CSS: when developer wants to style an element directly in the HTML file, less flexible.
  • external CSS: more common, keep all CSS rules in separate file, makes design changes easier.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can refer to a color by its hex number.
  • The period before "card" means it is a class selector, so any HTML element in that class will retain the property assignments.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swithces to it
  • add -A: commits all files in the working branch.
  • git pull origin main: makes sure our current branch is up to date, esp before opening a pull request.
